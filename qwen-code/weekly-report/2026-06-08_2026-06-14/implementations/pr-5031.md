## 实现目标

把 direct session shell 改成显式 opt-in 能力，降低 daemon 高风险命令入口默认暴露面。
PR body 只能辅助理解意图；本页的最终实现口径以 changed files、已提取的 patch 线索、测试和配置路径为准。

- 实现摘要：新增 `--enable-session-shell`，且必须同时配置 bearer token；REST/ACP/bridge 三层统一检查 enabled、token、session-bound client id，未启用时隐藏 capability 和 ACP method。
