## 解决的问题

避免 saved-session 回放时把已截断 edit/write diff 当作 raw output 再次回放，减少 web-shell 误渲染完整 diff 的风险。

## 最终实现方案

daemon replay 对截断 diff 只发 preview content，不再塞 raw output；web-shell transcript adapter 保留规范化 tool content，`ToolGroup` 忽略已截断 raw diff fallback，并在无 diff 时展示省略预览。
