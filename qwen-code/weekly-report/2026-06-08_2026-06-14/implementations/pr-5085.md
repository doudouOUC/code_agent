## 解决的问题

给 Agent/sub-agent 工具保留内部 `Kind.Agent` 分类，但避免把不存在的 ACP `agent` kind 发到线协议上导致 daemon Zod 校验丢帧。

## 最终实现方案

Agent 工具内部返回 `Kind.Agent`，`ToolCallEmitter.mapToolKind` 对外仍映射协议合法的 `other`；回退 WebUI/web-shell/Java SDK 等依赖 `agent` wire kind 的消费侧改动，锁定 no-regression 单测。
