## 最终实现方案

request helper 改用轻量子路径，新增 source import graph 与 bundle metafile 静态闭包回归测试。

## 关键代码路径

- `packages/cli/src/serve/server/request-helpers.ts`: serve request helper 改为轻量路径，避免静态导入 ACP runtime。
- `packages/cli/src/serve/fast-path.test.ts`: 新增 import graph 断言，保证 fast path source closure 不包含 ACP runtime。

## 验证方式

- `cd packages/cli && npx vitest run src/serve/fast-path.test.ts`: 验证 serve fast path 的 source import closure。
- 关键检查：`request-helpers.ts` 不再把 ACP/server runtime 作为 pre-listen 静态依赖。
