## 解决的问题

前台 shell 命令如果长时间没有输出，在 spawn 和 settle 之间不会发任何事件。TUI 还有 spinner，但 ACP gateway、`--output-format stream-json` 等 headless 消费者只能看到事件流静默，无法判断命令仍在运行还是执行链已经死掉。线上长任务诊断中，这类静默等待会被用户感知为 agent 卡住。

## 最终实现方案

PR 给 shell tool 增加静默命令活性心跳。`ShellProgressData` 加入 `ToolResultDisplay` union，字段包括 `type:'shell_progress'`、monotonic `elapsedMs`、可选 `lastOutputAgeMs`、`totalLines`、`totalBytes` 和 `timeoutMs`。它只携带统计信息，不携带命令输出，也不会进入模型上下文。配置链路新增 `tools.shell.heartbeatIntervalMs`，默认 10 秒，`0` 禁用；config 校验与 shell default timeout 一样限制为安全整数。

producer 在 shell process 真正 spawn 后启动 interval，使用 `performance.now()` 和最后一次真实输出时间判断 silent interval。data/binary progress 会刷新 last output time；abort、service throw、result settle 都清理 heartbeat timer。Scheduler 收到 `ShellProgressData` 时只转发给 `outputUpdateHandler`，不替换 accumulated live output，也不触发普通 display update。

消费者按职责分流。ACP `Session.runTool()` 把 heartbeat 转成 fire-and-forget meta-only `tool_call_update`，形如 `status:'in_progress'` 且 `_meta.shellProgress`；`toolSettled` gate 防止 heartbeat 与完成竞态导致 completed 后又出现 in_progress，并把 heartbeat count/last output age 写进 tool execution span。stream-json 通过已有 `tool_progress` pipeline 转发，受 `--include-partial-messages` 控制。TUI、subagent runtime、desktop `QwenAgent`、channel `DaemonChannelBridge` 和 web-shell normalizer 都识别并丢弃这些 meta-only frame，避免把心跳误渲染成 tool result 或覆盖 UI title。

## 关键代码路径

- `packages/core/src/tools/shell.ts`: heartbeat interval、silent detection、timer cleanup 和 `ShellProgressData` 生产。
- `packages/core/src/tools/tools.ts`: `ShellProgressData` 类型与 `isShellProgressData()` guard。
- `packages/core/src/core/coreToolScheduler.ts`: progress 转发但不覆盖 live output。
- `packages/cli/src/acp-integration/session/Session.ts`: ACP meta-only heartbeat frame、settle gate、span attributes。
- `packages/cli/src/nonInteractive/io/StreamJsonOutputAdapter.ts`: stream-json `tool_progress` variant。
- `packages/cli/src/config/settingsSchema.ts`, `packages/core/src/config/config.ts`: `tools.shell.heartbeatIntervalMs` 配置。
- `packages/channels/base/src/DaemonChannelBridge.ts`, `packages/sdk-typescript/src/daemon/ui/normalizer.ts`, `packages/desktop/packages/shared/src/agent/qwen-agent.ts`: 丢弃 meta-only heartbeat，避免 UI/protocol 状态回退。
