## 解决的问题

给 `qwen serve` 增加 ACP 权限/`ask_user_question` 单次响应超时配置，解决 5 分钟固定等待不可调的问题。

## 最终实现方案

新增 `--permission-response-timeout-ms`，从 CLI 透传到 `ServeOptions` / `createAcpSessionBridge` 覆盖 bridge 默认 5 分钟；`0` 表示无限等待；启动时拒绝非有限/负数/非整数，bridge 将超大值 clamp 到 Node timer 上限，补 serve/runQwenServe/server/bridge tests，已于 06-18 合入。
